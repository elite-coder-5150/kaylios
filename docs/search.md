## Documentation for the search class



The Search class is responsible for the following:

1. **Search by Id** - This is the most basic search. It will return a single result based on the id of the document.
2. **Search User** - This will return a list of users based on the search criteria.
3. **Search groups** - This will return a list of groups based on the search criteria.
4. **Search Dategory** - This will return a list of categories based on the search criteria.
5. **Tag Search** - This will return a list of tags based on the search criteria.
6. **Posts** - This will return a list of posts based on the search criteria.
